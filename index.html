Caelok UI Framework

A modern, lightweight, and customizable CSS framework, dynamically generated and injected via JavaScript for optimal performance and flexibility.

Getting Started

Integrating Caelok UI into your project is straightforward. You just need to include the framework.js script and optionally configure it.

1. Include the Script

Add the <script> tag to your HTML file, preferably before the closing </body> tag or in the <head> with the defer attribute.

<!-- In your HTML file -->
<script src="framework.js" defer></script>

2. Automatic Initialization

The framework can initialize automatically upon page load. If framework.js doesn't find a window.caelokConfig object, it will use its default settings.

3. Manual Initialization (with Options)

To customize the initialization (e.g., change the class prefix or initial theme), you can define a window.caelokConfig object before loading the script, or instantiate it manually after framework.js has loaded:

<!-- Option 1: Global Configuration (before framework.js) -->
<script>
  window.caelokConfig = {
    prefix: 'my-prefix-',
    theme: {
      colors: {
        primary: '#ff6347' // Tomato red
      }
    }
  };
</script>
<script src="framework.js" defer></script>

<!-- Option 2: Manual Instantiation (after framework.js is loaded) -->
<!-- <script src="framework.js"></script> --> <!-- Ensure framework.js is already loaded -->
<script>
  document.addEventListener('DOMContentLoaded', () => {
    // This script block runs for this demo page.
    // If window.Caelok wasn't auto-initialized by framework.js,
    // this page's script will attempt to initialize it with default settings.
    // For explicit custom configuration here:
    /*
    if (typeof CaelokFramework === 'function' && !window.MyFramework) {
        const customOptions = {
          prefix: 'app-',
          theme: {
            colors: {
              primary: '#3498db' // Peter River blue
            }
          }
          // ... more options
        };
        window.MyFramework = new CaelokFramework(customOptions);
        // Make sure to use window.MyFramework instead of window.Caelok if you do this.
    }
    */
  });
</script>

Once initialized, the framework will inject the necessary styles into the document's <head>, and the global variable (window.Caelok by default) will be available to interact with its API.

Layout System

Caelok UI uses a powerful Flexbox-based grid system to build responsive layouts efficiently.

Containers

Use .ck-container for a responsive max-width container or .ck-container-fluid for a full-width (100%) container.

.ck-container (responsive fixed-width)
.ck-container-fluid (full-width)
<div class="ck-container">...</div>
<div class="ck-container-fluid">...</div>

Rows and Columns

Rows (.ck-row) are wrappers for columns. Columns (.ck-col-*, .ck-col-<bp>-*) define the width and layout.

Column 1 (md-6, lg-4)
Column 2 (md-6, lg-4)
Column 3 (lg-4)
<div class="ck-row">
  <div class="ck-col-12 ck-col-md-6 ck-col-lg-4">...</div>
  <div class="ck-col-12 ck-col-md-6 ck-col-lg-4">...</div>
  <div class="ck-col-12 ck-col-lg-4">...</div>
</div>

Spacing (Margins & Paddings)

Use .ck-m-* for margins and .ck-p-* for paddings. Suffixes indicate side (t, b, s, e, x, y) and size (0-9).

Padding 3 (.ck-p-3)
Margin Top 4 (.ck-mt-4)
<div class="ck-p-3">...</div> <!-- padding: var(--ck-spacing-3) -->
<div class="ck-mt-md-4">...</div> <!-- margin-top on md breakpoint and up -->

Iconography

Caelok UI makes it easy to insert SVG icons. Use the Caelok.getIcon(iconName, attributes) API.

<!-- In your HTML -->
<span id="icon-container"></span>

<!-- In your JavaScript -->
<script>
  document.addEventListener('DOMContentLoaded', () => {
    if (window.Caelok) {
      const iconContainer = document.getElementById('icon-container');
      const myIcon = window.Caelok.getIcon('check', { width: '24', height: '24', class: 'ck-text-success' });
      if (myIcon) {
        iconContainer.appendChild(myIcon);
      }
    }
  });
</script>

Default available icons include: arrow-right, check, x, menu, search. You can add more to the framework's configuration.

Customization & Theming

Caelok UI is designed for easy customization through CSS Custom Properties and a theming API.

CSS Variables

Modify global CSS variables to change the appearance. For example, to change the primary color:

:root {
  --ck-theme-colors-primary: '#YOUR_NEW_COLOR_HERE';
  --ck-typography-fontFamilyBase: 'Your New Font, sans-serif';
  /* ...and many more variables! */
}

updateTheme(newTheme) API

Change the theme dynamically using JavaScript:

window.Caelok.updateTheme({
  colors: {
    primary: '#e83e8c', // Pink
    bodyBg: '#333',
    textColor: '#f8f9fa'
  },
  typography: {
    fontFamilyBase: 'Georgia, serif'
  }
});
